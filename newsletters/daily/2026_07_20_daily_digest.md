<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Claude Fable 5 disproves Jacobian conjecture — 3D polynomial counterexample</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Anthropic mathematician Levent Alpöge posts an explicit polynomial map that breaks a central 87-year-old conjecture in algebraic geometry, built with the help of Claude Fable 5.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
The Jacobian conjecture, posed by Ott-Heinrich Keller in 1939, asks whether any polynomial map from n-dimensional complex space to itself with a non-zero constant Jacobian determinant must have a polynomial inverse. Levent Alpöge — a Harvard Junior Fellow now at Anthropic — posted an explicit 3D polynomial map with Jacobian determinant -2 that sends three distinct points to the same image, disproving it outright.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Alpöge built the counterexample with Claude Fable 5, which helped generate and test candidate polynomial maps. Verification is direct arithmetic: the Jacobian determinant evaluates to -2 everywhere, and three named points all map to the same image, proving the map is not invertible.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The Jacobian conjecture resisted proof and counterexample since 1939, making this a landmark algebraic-geometry result. It follows OpenAI's May 2026 disproof of Erdős's unit-distance conjecture, signaling that frontier general-purpose reasoning models are now routine tools for attacking open mathematical problems.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Algebraic geometers, AI-for-math watchers, and mathematicians using LLMs as research collaborators.
</p>
