<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">NVIDIA SpatialClaw — code as the action interface for spatial reasoning agents</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">NVIDIA framework that lets a spatial-reasoning agent write Python each turn instead of picking from a fixed tool menu.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
<a href="https://arxiv.org/abs/2606.13673" style="color: #f7ff00;">SpatialClaw</a> is an NVIDIA Research paper and open code release for vision-language agents that answer spatial questions from images and video — "is the cup behind the laptop?" or "how far is the chair from the door?" — by writing executable Python each turn.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The agent runs inside a persistent Python kernel pre-loaded with perception primitives (Depth Anything 3, SAM 3, NumPy, SciPy). Each step is a five-stage loop — plan, generate code, execute, read result, decide — keeping earlier computations in kernel state for later steps.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Treating code as the action interface lifts average accuracy by +11.2 points across 20 spatial benchmarks over prior SOTA, with the biggest gains on video and multi-view tasks — and works across six VLM backbones with no per-model tuning.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
VLM researchers and robotics engineers working on spatial reasoning agents.
</p>
