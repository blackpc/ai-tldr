<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Claude Fable 5 redeployed — Anthropic ships globally after US lifts export controls</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Claude Fable 5 goes global again on July 1, this time behind a defense-in-depth safety stack.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Claude Fable 5 is Anthropic's general-purpose flagship, paused in June after a jailbreak went public. US export controls came off on June 30, and global rollout to Claude.ai, Claude Code, Claude Cowork, and the Claude Platform started July 1.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Anthropic layered four defenses on the relaunched model: safety training, retroactive misuse pattern analysis, cybersecurity classifiers, and a new targeted classifier that blocks the specific jailbreak in over 99% of tested cases.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Fable 5 access unblocks the daily-driver Claude experience for millions of Pro, Max, Team, and Enterprise users — and sets a template for how frontier labs can pause, patch, and return a model under an audited safety stack.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Claude users on Pro, Max, Team, and Enterprise plans (up to 50% of weekly limits through July 7), plus developers on the Claude API and Claude Code users who lost Fable 5 access in June.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">xAI's Voice Agent Builder is a no-code way to ship production voice agents on Grok Voice, priced by the minute.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
<a href="https://x.ai/news/grok-voice-agent-builder" style="color: #f7ff00;">Voice Agent Builder</a> is xAI's no-code console for wiring up production voice agents on top of the Grok Voice model. Pick a voice, add tools and a knowledge base, hook up telephony, and have a running agent in about two minutes — no glue code between STT, LLM, and TTS.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>

</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">GeneBench-Pro — OpenAI's 129-problem computational-biology benchmark</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">A research-level bio benchmark where top frontier models still fail more than two thirds of the time.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
<a href="https://huggingface.co/datasets/ajh-oai/genebench-pro-public-package" style="color: #f7ff00;">GeneBench-Pro</a> is a 129-problem benchmark from OpenAI that hands an AI agent synthetic-but-realistic tasks from genomics, quantitative biology, and translational medicine — targeting real research judgment, not exam-style recall.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The agent gets data files, a Python + PLINK 2.0 workspace, and a target estimand; every answer is graded against a known ground truth. GPT-5.6 Sol Pro tops the leaderboard at 31.5%; Claude Opus 4.8 lands second at 16.0%.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Coding evals are saturating — GeneBench-Pro reopens the gap. With 82 of 129 tasks vetted by grad students, postdocs, and industry scientists, scores here map to real research competence rather than benchmark gaming.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
AI evaluation researchers and bio/AI labs; a 10-question public subset is available on Hugging Face under CC-BY-4.0.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">DeepSeek V4 gets peak-hour pricing — API doubles 9am–12pm and 2pm–6pm Beijing time</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">First large LLM API to introduce time-of-day pricing.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
<a href="https://api-docs.deepseek.com/quick_start/pricing" style="color: #f7ff00;">DeepSeek V4</a> will charge double during two Beijing-time windows — 9 AM to noon and 2 PM to 6 PM — on both V4 Pro and V4 Flash, taking effect when V4 officially launches in mid-July 2026.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The API splits the day into peak (2×) and off-peak (1×) blocks in Beijing local time, seven days a week. Both output and cache-miss input tokens double at peak; users get an email 24 hours before their meter switches.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
This is a first for any major LLM API — mirroring how power grids handle demand. Teams running batch workloads outside peak windows (before 9 AM, 12–2 PM, or after 6 PM Beijing time) can cut costs in half; expect other providers to take note.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Teams running production DeepSeek V4 workloads — especially those with schedulable batch jobs like nightly code review, document ingest, or offline evaluations.
</p>
