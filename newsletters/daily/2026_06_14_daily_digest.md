<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Anthropic Suspends Claude Fable 5 and Mythos 5 — US national-security order</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Anthropic must disable both flagship models after a US export-control order tied to a narrow jailbreak finding.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
On June 12, Anthropic received a US government directive ordering it to suspend access to Claude Fable 5 and Claude Mythos 5 for any foreign national — and to comply, disabled both models for every customer worldwide.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The order cites national security. Anthropic says the trigger was a narrow potential jailbreak in Fable 5 when the model reviews a codebase for software vulnerabilities; its other Claude models, including Opus 4.8, stay available.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Every customer calling the Fable 5 or Mythos 5 API sees requests fail. It sets a first-of-its-kind precedent for how US export controls can be applied to a live commercial AI product over a single capability concern.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Anthropic API customers and Claude developers who need to roll back to <a href="https://www.anthropic.com/claude" style="color: #f7ff00;">Claude Opus 4.8</a> immediately.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Google Sues 'Outsider Enterprise' — Chinese scam ring abused Gemini</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">First time Google has gone to court over Gemini abuse — and the defendant is a phishing-as-a-service ring.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
Google filed a civil lawsuit against the 'Outsider Enterprise,' a China-based cybercrime network that sells phishing kits on Telegram and used <a href="https://deepmind.google/technologies/gemini/" style="color: #f7ff00;">Gemini</a> to auto-generate fake login pages for Google, YouTube, USPS, and other trusted brands.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Outsider sells 290+ phishing templates for $88–$200/week. Members asked Gemini to build 'gift redemption' pages framed as innocent requests, fed the generated code into Outsider's platform to spin up scam sites, and blasted 2.5 million SMS messages to Android users in a two-week stretch.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
It is the first time a major AI provider sued an end-to-end abuse network built on its own model — setting a precedent that providers will pursue users who dress malicious prompts as innocent code requests.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
AI safety teams, trust-and-safety engineers, and security researchers watching how providers respond to model misuse at scale.
</p>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">TensorZero archives its repo — open-source LLMOps gateway winds down</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">An 11.6k-star Rust LLM gateway shuts down 8 months after seed, leaving Apache-2.0 code in place but unmaintained.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
<a href="https://www.tensorzero.com" style="color: #f7ff00;">TensorZero</a>, an open-source LLMOps platform written in Rust that bundled an LLM gateway with multi-provider routing, observability, evaluation, and experimentation, has archived its GitHub repo and wound down operations.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The company raised a $7.3M seed in 2024, spent less than half, then returned the rest to investors after co-founder Gabriel Bianconi said they needed to find product-market fit twice — once for the OSS, once for the commercial layer — in a market that shifted under them.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
With no upstream patches or new provider integrations coming, teams running TensorZero need to plan a migration now — and the shutdown makes open-source LLMOps as a venture-backed category harder to justify.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Teams self-hosting LLM gateways or evaluating open-source LLMOps tooling who need to find a replacement.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">agent-skills 0.6.2 — Addy Osmani adds /build and /webperf for AI coding agents</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Addy Osmani's open-source agent-skills bundle hits v0.6.2 with new slash commands and hardened security skills.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
<a href="https://github.com/addyosmani/agent-skills" style="color: #f7ff00;">agent-skills</a> is a collection of 24 structured workflows you load into a coding agent like <a href="https://claude.ai/code" style="color: #f7ff00;">Claude Code</a>, Cursor, Codex, or Gemini CLI — each a Markdown file encoding a senior engineer's checklist for TDD, security review, and performance audits.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
v0.6.2 adds a <code style="font-family: Menlo, Consolas, monospace; font-size: 13px; color: #f7ff00;">/build</code> one-pass plan-and-implement command, a <code style="font-family: Menlo, Consolas, monospace; font-size: 13px; color: #f7ff00;">/webperf</code> auditor, an observability skill, native Antigravity CLI support, and stronger security skills covering threat modeling, SSRF, and LLM-specific issues.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Coding agents skip tests and ship quick patches unless you stop them — agent-skills puts that brake in front of the model, now covering 24 workflows across 56.7k GitHub stars.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Engineers using Claude Code, Cursor, Codex, Gemini CLI, or Antigravity CLI who want structured guardrails on their coding agent — install via <code style="font-family: Menlo, Consolas, monospace; font-size: 13px; color: #f7ff00;">/plugin marketplace add addyosmani/agent-skills</code>.
</p>
