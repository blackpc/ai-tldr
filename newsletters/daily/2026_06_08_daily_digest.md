<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Google Ships the Colab CLI — Apache-2.0 Tool Provisions A100 and H100 GPUs From Your Terminal, With an Agent Skill for Claude Code and Codex</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Open-source CLI that hands local terminals — and any coding agent — Colab GPUs and TPUs on demand.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
An Apache-2.0 command-line client for <a href="https://colab.research.google.com" style="color: #f7ff00; text-decoration: none;">Google Colab</a> that drives Colab GPU/TPU runtimes from a normal shell instead of the notebook UI. It also ships a prebuilt skill file so any agent with terminal access — Antigravity, <a href="https://claude.ai/code" style="color: #f7ff00; text-decoration: none;">Claude Code</a>, Codex — can call it directly.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
<code style="font-family: Menlo, Consolas, monospace; font-size: 13px; color: #f7ff00;">colab new --gpu A100</code> provisions a runtime in seconds. <code style="font-family: Menlo, Consolas, monospace; font-size: 13px; color: #f7ff00;">colab exec -f script.py</code> runs a local file on the remote VM. Supported accelerators include T4, L4, G4, A100, and H100 GPUs plus v5e1 and v6e1 TPUs; artifacts and logs come back with <code style="font-family: Menlo, Consolas, monospace; font-size: 13px; color: #f7ff00;">colab download</code>.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
An agent on a laptop can now ask for a fresh H100, run a fine-tune or eval, pull the logs, and tear down — all through the same shell tools it already uses. Until now, hooking Colab into an agent loop required scripting the notebook UI or wiring the open-source Colab MCP server.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
AI agents and developers who want one-line access to Colab GPUs/TPUs from a terminal without touching the notebook interface.
</p>

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">OpenAI, Anthropic, Google DeepMind, and Microsoft AI CEOs Co-Sign Letter Urging Congress to Mandate DNA Synthesis Screening</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">The four frontier-lab CEOs put their names on the same page for once — telling Congress to screen synthetic-DNA orders before AI does it for the wrong people.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>

</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The letter asks for two requirements: orders screened against curated sequences-of-concern databases, and provider recordkeeping of all customers and orders. A companion bill from Senators Cotton and Klobuchar — <a href="https://www.cotton.senate.gov/news/press-releases/cotton-klobuchar-introduce-bill-to-establish-federal-biotech-security-framework" style="color: #f7ff00; text-decoration: none;">S.3741</a> — would codify this plus a NIST biotech governance sandbox.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The signatories warn that LLMs are eroding the knowledge barrier that has historically kept biological weapons hard to design. Getting Altman, Amodei, Hassabis, and Suleyman to co-sign anything is a signal in itself — they are direct competitors who have rarely agreed on AI policy, and nucleic-acid screening is the chokepoint between an AI-designed sequence and a real-world pathogen.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Biosecurity policy staff, frontier-lab government-affairs teams, gene-synthesis providers, and congressional offices tracking AI-bio risk.

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">S&P Dow Jones Indices keeps the seasoning, float, and profitability bars intact, so OpenAI and Anthropic must wait at least a year after IPO before joining the index.</p>

AI-IPO watchers, capital-markets analysts, and passive-fund LPs tracking when and how AI lab equity enters benchmark indices.

<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">Lathe v0.3.0 — Open-Source CLI Uses Claude Code, Cursor, and Codex to Generate Hands-On Tutorials, Served Locally at localhost:4242</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Generate hands-on tutorials with an agent skill, then read and type the code yourself in a local UI built for it.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
<a href="https://github.com/devenjarvis/lathe" style="color: #f7ff00; text-decoration: none;">Lathe</a> is a Go CLI plus an LLM agent skill that turns "teach me X" prompts into multi-part, source-backed tutorials. Install it from Homebrew, then run <code style="font-family: Menlo, Consolas, monospace; font-size: 13px; color: #f7ff00;">/lathe build a 3D slicer in Erlang</code> from <a href="https://claude.ai/code" style="color: #f7ff00; text-decoration: none;">Claude Code</a>, Cursor, or Codex, and Lathe assembles a structured course with exercises into a local store.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
The skill instructs the agent to scaffold lessons as markdown with embedded exercises, links to primary sources, and verification steps. <code style="font-family: Menlo, Consolas, monospace; font-size: 13px; color: #f7ff00;">lathe serve</code> boots a local web UI at http://localhost:4242 with search, table of contents, and progress tracking — the human reads the material and re-types the code themselves.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
Most agent skills automate work away from the developer. Lathe inverts the model: the LLM curates and scaffolds, the human does the typing and thinking — which preserves the muscle memory that comes from actually working through an unfamiliar stack.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Engineers learning a new stack who want curated, hands-on study materials instead of a chatbot Q&amp;A session.
</p>
