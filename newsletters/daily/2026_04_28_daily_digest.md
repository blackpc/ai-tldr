<h2 style="margin: 0 0 8px; font-size: 24px; font-weight: 800; color: #f5f5f0; line-height: 1.2;">OpenAI Shuts Down Sora — $1M/Day Video AI Discontinued After 17 Months</h2>

<p style="margin: 0 0 20px; font-size: 15px; color: #8a8a85; line-height: 1.5;">Sora's text-to-video app closes after 17 months — high compute costs, weak retention, and a strategic shift toward coding tools ended it.</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">What is it?</strong><br>
OpenAI's Sora text-to-video app was shut down on April 26, 2026, 17 months after its December 2024 launch. Active users had declined to under 500,000, and the product was losing an estimated $1 million per day. The Sora API will close on September 24, 2026.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">How does it work?</strong><br>
Sora used a video diffusion transformer architecture that required substantially more compute than text or image generation, making per-generation costs uneconomical at consumer price points. Physics failures and anatomy errors in generated video further limited practical adoption.
</p>

<p style="margin: 0 0 16px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Why does it matter?</strong><br>
The shutdown signals OpenAI pivoting engineering and compute resources toward ChatGPT coding tools and enterprise products. A next-generation replacement codenamed Spud is in development, targeting the physics consistency and prompt-accuracy issues that undermined Sora.
</p>

<p style="margin: 0 0 20px; font-size: 15px; color: #f5f5f0; line-height: 1.6;">
<strong style="color: #f7ff00;">Who is it for?</strong><br>
Developers and organizations who built workflows on Sora — and teams evaluating AI vendor platform durability before committing to API-dependent pipelines.
</p>
